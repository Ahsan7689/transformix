## рҹҡҖ Getting Started

### Prerequisites

- **Node.js 18+** вҖ” [Download here](https://nodejs.org)
- A **[Supabase](https://supabase.com)** project (free tier works perfectly)

### 1 В· Clone & Install

```bash
git clone https://github.com/YOUR_USERNAME/transformix.git
cd transformix
npm install
```

### 2 В· Environment Variables

Create a `.env` file in the root:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

> рҹ”‘ Find these in your Supabase project under **Settings вҶ’ API**

### 3 В· Database Setup

Run this SQL in your **Supabase SQL Editor**:

```sql
-- Credits table
create table credits (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id) on delete cascade,
  ip_address text,
  credits integer not null default 25,
  last_reset timestamptz not null default now(),
  created_at timestamptz default now()
);

-- Short links table
create table short_links (
  id uuid primary key default gen_random_uuid(),
  code text unique not null,
  original_url text not null,
  clicks integer default 0,
  created_at timestamptz default now()
);

-- Enable Row Level Security
alter table credits enable row level security;
alter table short_links enable row level security;

-- Policies
create policy "Users manage own credits" on credits
  for all using (auth.uid() = user_id or user_id is null);

create policy "Anyone can read short links" on short_links
  for select using (true);

create policy "Anyone can create short links" on short_links
  for insert with check (true);

create policy "Anyone can update clicks" on short_links
  for update using (true);
```

### 4 В· Run Locally

```bash
npm run dev
# Open вҶ’ http://localhost:5173
```

## рҹӘҷ Credit System

| User Type | Starting Credits | Refill Amount | Refill Every |
|:---:|:---:|:---:|:---:|
| рҹ‘Ө Guest | **25** | +10 | 28 hours |
| вң… Signed In | **100** | +50 | 24 hours |

Credits persist per-user via `localStorage` using unique keys вҖ” they **never reset on page refresh** and are completely isolated between users:

```
Guest user   вҶ’  localStorage key: tx_credits_guest
Logged in    вҶ’  localStorage key: tx_credits_user_[userId]
```

| рҹ§  **Model** | Flux (state of the art, 2026) |

Images are proxied through Vite in development to avoid CORS issues:

```js

proxy: {
  '/api/image': {
    target: 'https://image.pollinations.ai',
    changeOrigin: true,
  }
}
```
